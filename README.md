📊 Airbnb NYC Dashboard
📌 Project Overview

This project is an interactive Power BI dashboard built to analyze Airbnb listings in New York City. It helps in understanding pricing trends, room type distribution, neighbourhood performance, and customer preferences.

🎯 Objectives

Analyze average price across neighbourhoods
Understand distribution of room types
Identify most common property types
Track total listings and reviews
Provide insights for better decision-making

📈 Key Insights
Manhattan has the highest average prices
Entire home/apartment is the most popular room type
Apartments dominate property listings
Pricing varies significantly across different neighbourhoods

📊 Dashboard Features
Average price by neighbourhood
Total listings by room type
Average price by room type
Total listings by property type
KPI cards (Total Listings, Average Price, Reviews, Review Score)
Interactive filters for neighbourhood and room type

🛠 Tools & Technologies
Power BI
Power Query (Data Cleaning)
DAX (Data Analysis Expressions)

📂 Dataset
Airbnb NYC dataset (public dataset)

🚀 How to Use
Download the .pbix file
Open in Power BI Desktop
Explore the dashboard using filters and visuals
